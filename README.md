# Task-Manager-FullStack

Full-Stack CRUD Rakenduse Harjutus
Ülesande kirjeldus

Arenda täisfunktsionaalne Full-Stack CRUD (Create, Read, Update, Delete) rakendus, kus:

Backend on loodud Java (Spring Boot) või C# (ASP.NET Core) abil.
Frontend on loodud vabalt valitud raamistikuga (React, Vue, Angular, Svelte vms).
Andmed salvestatakse andmebaasi (nt PostgreSQL, MySQL või SQL Server).
Frontend suhtleb backendiga REST API kaudu.
Teha uus repositoorium GitHubi. Iga väiksema ülesande juures tuleb teha commit ja push.
Koodi üles laadimise nimetused peavad olema tähendusega. Mitte asd, update, doSomething.

Rakendus peab võimaldama kasutajal:

Lisada uusi kirjeid.
Vaadata olemasolevaid kirjeid.
Muuta olemasolevaid kirjeid.
Kustutada kirjeid.
Kuvada andmeid kasutajasõbralikus vaates.

Kohustuslikud nõuded
Backend
REST API, CRUD operatsioonid, andmebaasi kasutamine, sisendi valideerimine, korralik projektistruktuur

Frontend
Kirjete loetelu vaade, uue kirje lisamise vorm, muutmise vorm, kustutamise võimalus, veateadete ja edukate tegevuste kuvamine

Andmebaas
Vähemalt üks tabel, primaarvõti, vähemalt 5 sisulist välja

Teema:
Ülesannete haldur (Task Manager)
